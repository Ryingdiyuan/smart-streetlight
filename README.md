# 智慧路灯节能系统

本仓库用于重庆交通大学-中软国际企业实训项目，选题为“智慧路灯节能系统”。

项目目标是完成一个端-智-云一体化应用：

- 校内阶段：使用模拟设备产生光照和设备状态数据，通过 MQTT 上报后端，前端展示实时数据、历史趋势和告警信息，并接入智能体维护问答。
- 基地阶段：接入鸿蒙开发板、光照传感器和路灯控制外设，将模拟数据链路切换为真实硬件链路。
